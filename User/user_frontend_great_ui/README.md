# User Frontend (Great UI)

React + TypeScript + Tailwind UI for the **User** project.

## Install & Run
```bash
npm i
npm run dev
```

Point it at your User backend:
- Create `.env` and set `VITE_API=http://localhost:8002` (optional; default is that)
- Login with a seeded user (e.g., `alice@example.com / pass123`)

## Pages
- Users (cards)
- Templates (chips)
- Rate (slider-based per field, 0–100)
- History (timeline style)
- Leaderboard (bar chart)
